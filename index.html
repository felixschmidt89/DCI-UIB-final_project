I've spent my professional life (7 years) in different roles centered around WEB software at dynamic, fast-growing and ever-changing tech startups. Project

This website is my final project of the very first technical module as part of the Web Development Course at Digital Care Insitute (DCI): During this module (User Interface Basic) I’ve learned to design & build websites with HTML & CSS first and then expanded on the basics of Sass and bootstrap afterwards. Now, it's time to showcase what I've learned up to this point.
After dabbling with Bootstrap and stumpling upon its limitations regarding some of my ideas and also given the scope and purpose of this project, I’ve decided against using Bootstrap and build everything from scratch without using any framework or template. But since I'm looking forward to collaborating with others and value structured and DRY coding, I've made use of Saas. If you'd like to see my code, please refer to the related GitHub repository. If you want to learn more about what other skills I'm about to acquire during the course at DCI, please give this a read. Next up will be JavaScript & React.
